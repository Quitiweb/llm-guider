# llm-guider

La idea inicial es hacer una web que guíe a los usuarios a usar la IA para sus propósitos.

## 💡 Idea general

**llm-guider** es una plataforma web que ayuda a cualquier persona —sin importar su nivel técnico— a sacar el máximo partido de los modelos de lenguaje (LLMs) para sus propias necesidades.

El objetivo es eliminar la barrera de entrada que supone no saber *cómo preguntar* a una IA, ofreciendo guías, plantillas y flujos de trabajo adaptados a distintos casos de uso.

---

## 🎯 Objetivos principales

- **Guiar** al usuario paso a paso para formular prompts efectivos.
- **Categorizar** los casos de uso más comunes (escritura, código, estudio, trabajo, creatividad, etc.).
- **Ofrecer plantillas** de prompts reutilizables y editables.
- **Enseñar** buenas prácticas en el uso de IAs generativas.
- **Ser accesible** para usuarios no técnicos.

---

## 🗂️ Casos de uso iniciales contemplados

| Categoría       | Ejemplos de uso                                     |
|-----------------|-----------------------------------------------------|
| 📝 Escritura     | Redactar correos, artículos, resúmenes              |
| 💻 Código        | Generar, explicar y depurar código                  |
| 🎓 Estudio       | Crear apuntes, resúmenes, preguntas de repaso       |
| 💼 Trabajo       | Informes, presentaciones, análisis de datos         |
| 🎨 Creatividad   | Brainstorming, historias, guiones                   |
| 🔍 Investigación | Búsqueda de información, comparativas, traducciones |

---

## 🛠️ Stack tecnológico (ideas preliminares)

- **Frontend:** React / Next.js
- **Backend:** Python (FastAPI o Django)
- **IA:** Integración con APIs de LLMs (OpenAI, Anthropic, modelos open-source, etc.)
- **Base de datos:** PostgreSQL para guardar plantillas y progreso del usuario
- **Despliegue:** Docker + plataforma cloud (por definir)
